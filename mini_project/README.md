# Expense tracker(CLI Project)

## description
a simple command - line based expense tracker built using python . it allows users to add expenses , view all expenses , and calculate total spending.
Data is stored permanently using a file.

## Features 
- menu-based CLI
-add expense with date 
-view all expenses 
- calculate total expense
-file handling
-OOP- based design
-input validation

## how to run
1. navigate to the mini_project folder
2. Run the program:
python expense_tracker.py

## File storage 
all expenses are saved in "expense.txt"
